# Code Generator This web application allows you to generate code snippets in various programming languages based on your input text and options. ## Technologies Used - HTML5 - CSS3 - JavaScript - Font Awesome (for icons) - Responsive Design Principles ## Contributors - [@YourGitHubUsername](https://github.com/GTK-THANGELLA-17) - G.Thangella ## Links - [Live Demo](https://gtk-thangella-17.github.io/code-generator-v1/) - Check out the live demo of the website. ## How to Use 1. **Enter Text:** Input your desired text in the "Enter Text" field. 2. **Enter Link:** Optionally, input a link in the "Enter Link" field. 3. **Upload File:** Optionally, upload a file by clicking on the "Upload File" button. 4. **Select Programming Language:** Choose the programming language for which you want to generate code from the dropdown list. 5. **Generate Code:** Click on the "Generate Code" button to generate the code snippet based on your input. 6. **View Generated Code:** The generated code will be displayed in the code output section. 7. **Copy Code:** Click on the "Copy Code" button to copy the generated code to your clipboard. ## Supported Programming Languages - HTML - CSS - JavaScript - React - Angular - Swift - C - C++ - Java - Python - C# - Go - Kotlin - PHP - R - Ruby - Matlab - TypeScript - Scala - SQL - NoSQL - Rust - Perl ## Credits - Prism.js for syntax highlighting. - Designed and built by G.
